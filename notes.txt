Comienzo

-Para usarlo, corremos el comando: npx create-next-app [nombre-app].
-Esto nos genera un proyecto completo.

-Automaticamente vienen los script start, dev y build.

Cosas importantes que se crearon:
  -pages/api: Un ejemplo de las rutas que tendrá la API y sus endpoints.
  -pages/index.js: Core de nuestra app, la ruta principal.
  -public: Subiremos imagenes, estaticos, y lo que sea público.

-Al correr dev, nos crea una carpeta con estaticos, .next.

Styled JSX
  -Lo que haremos será, dentro de la etiqueta que encapsula todo el
    componente, colocamos una etiqueta <style jsx> la cual, tendrá
    por dentro unas comillas invertidas (para casos donde haya que
    usar variables).
  -Dentro de esa comilla se colocarán los estilos que podrían ir en
    el css.
  -Esto tiene una particularidad, que al usar un selector de 
    elemento, por ejemplo h1, sí hubiera h1 en otras páginas, esos
    estilos no se aplican a esas páginas.
  -Esto porque al agregar estilos de esta forma, lo que hace es crear
    una clase especifica para ese componente.
  -Recordar que las comillas deben ir dentro de {}.
  -Tambien se pueden usar className, lo cual es bueno hacerlo.
  -Estilos globales: Para ello, creamos una etiqueta similar,
    pero con la prop 'global'. 
  -Podemos tener varias en el mismo componente.
