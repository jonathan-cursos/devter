Obtener los datos de firestore

-Tener presente que cada que creamos un documento en la colección, nos genera
  automaticamente un ID de ese documento.

Obtener datos de firestore:
  -Creamos una funcion en firebase/client, que retorne los metodos 
    .collection('nombre_coleccion').get() de la conexión a la DB.
  -Esto retorna una promesa, esta no retornará los datos, sino un elemento 
    'snapshot', este tiene un array con todos los documentos.
  -De cada documento podemos sacar varias cosas, entre ellas:
    .data(): Esto retorna el objeto con cada dato de un documento.
    .id: Este es el id de cada archivo (no del usuario de GH).
  -Finalmente retornaremos un objeto, con los datos necesitados sacados de cada
    archivo.
  -Al usar esta función, nos retornará una promesa con ese objeto.
